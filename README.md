# Archer Configuration Toolkit

Archer Configuration Toolkit is a modular and easy way to setup and configure different parts of your Arch Linux system

Select one or more arrows from the quiver and Archer Configuration Toolkit will guide you to install and configure the selected features for your system

Run `bash <(curl -L https://raw.githubusercontent.com/Martan404/Archer-Config/main/archer)` in your terminal to get started
